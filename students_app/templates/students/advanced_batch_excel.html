Excel File Format - Flexible Column Structure
You decide what fields to include! Add only the columns that match your ID card template. The system will automatically detect common field names.
How It Works:
  1. Include columns for the fields you want on your ID card
  2. Use any column names you prefer (system auto-detects common names)
  3. Configure field mappings in the template editor if needed
  4. System will map Excel columns to ID card fields automatically
Common Fields (Auto-Detected):
  • Name: Name, Student Name, F. Name, full_name
  • ID/Admission: ID, Admission Number, Student ID
  • Class: Class, Class Name, Grade
  • Section: Section, section
  • Contact: Mobile, Phone, Contact
  • DOB: DOB, Date of Birth, Birth Date
  • Address: Address, Home Address
  • Photo: Photo, Photo No, Photo Number
Photo Mapping: Use photo numbers in Excel (1, 2, 3, etc.) and name photos in ZIP file as 1.jpg, 2.jpg, 3.jpg, etc. The system will automatically map the photo number to the corresponding photo file.
Custom Field Names? You can configure field mappings in the Template Editor to map your custom column names to ID card fields.

Need Help?

  • Include columns only for fields you want to display
  • Column names are case-insensitive (Name = name = NAME)
