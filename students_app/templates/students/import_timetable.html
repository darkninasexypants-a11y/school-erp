Supported formats: .xlsx, .xls
Excel File Format
Required Columns:
  • Section - Format: "Class Name - Section Name" (e.g., "UKG - A", "Class 1 - B", "Class 10 - A")
    Note: Use exact class and section names as they appear in your system
  • Day - Day of week (Monday, Tuesday, Wednesday, Thursday, Friday, Saturday)
  • Subject - Subject name (will be created automatically if it doesn't exist)
Optional Columns:
  • Time Slot - Time slot name (must match existing time slots)
  • Teacher - Teacher name (will match by first name)
  • Room - Room number
Example Excel Format:
Section Day Time Slot Subject Teacher Room
UKG - A Monday Period 1 Mathematics John Doe 101
UKG - A Monday Period 2 English Jane Smith 102
Class 1 - A Tuesday Period 1 Science Robert Johnson 103
Section Format: Use "Class Name - Section Name" (e.g., "UKG - A", "Class 1 - B", "Class 10 - A"). Make sure the class and section names match exactly with your system.
Note:
  • Sections must already exist in the system
  • Subjects will be created automatically if they don't exist
  • Time slots must match existing time slots in the system
  • Teachers will be matched by first name if found
  • Existing entries will be updated if they match (same section, day, time slot)
